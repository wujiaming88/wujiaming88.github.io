# 博客文章配图规范

## 原则
1. 每篇文章使用**固定**的 Unsplash 图片，不使用随机图片
2. 图片主题必须与文章分类/标签相符
3. 使用 Unsplash 固定 URL 格式，确保图片不会变化

## URL 格式
```
https://images.unsplash.com/photo-{id}?w=1200&h=400&fit=crop
```

## 新文章配图流程
1. 确定文章的主要分类/标签
2. 从上方映射表中选择匹配的图片 ID
3. 尽量避免与最近的文章使用相同图片
4. 如需新图片，去 unsplash.com 搜索相关关键词，获取 photo ID
5. URL 格式：`https://images.unsplash.com/photo-{id}?w=1200&h=400&fit=crop`
6. 写入 front matter 的 `header.overlay_image` 字段
