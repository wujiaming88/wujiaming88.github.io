## 二、验证配置

### 1. 验证DNS解析
配置完成后，可以使用以下命令验证解析是否生效：
```bash
nslookup blog.juanwang.tech
# 或
dig blog.juanwang.tech
```

### 2. 验证GitHub Pages部署
等待10-30分钟后，访问：`https://blog.juanwang.tech`

如果看到你的博客内容，说明配置成功！

## 三、常见问题

### 1. 配置后无法访问
- 请等待DNS解析生效（通常需要10-30分钟）
- 检查DNS记录是否正确
- 检查GitHub Pages设置是否已启用自定义域名

### 2. HTTPS证书问题
GitHub Pages会自动为自定义域名颁发HTTPS证书，通常需要24小时内生效。如果访问时出现证书错误，请耐心等待或刷新页面。

### 3. 域名所有权验证
如果GitHub Pages提示需要验证域名所有权，请按照提示添加TXT记录。
